Llenguatge de compilacio presentat per el Grup 2, assignatura Lenguatges de compilació.

Per tal d'executar el programa, cal:

	- Tenir IntelliJIdea per poder executar el codi Java.
	- Tenir una versió de Java 11 o superior (la que s'ha usat per la programació).

Per tal d'executar els diversos tests que es disposen, descomentar el text desitjat en el Main.java, linies 16 a 35. Es poden executar mes d'un test si aquests no disposen de cap error. El programa tallara la execucio en cas de trobar-se un error.

Si el codi fos font de problemes, es recomana usar la ultima versio estable disponible en el GitHub del equip: https://github.com/ls31300/Llenguatges
